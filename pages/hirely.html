Hirely

| Personal

Built a Flutter-based app to connect users with nearby laborers, with backend services in Node.js/MongoDB. Integrated Firebase and Google Places API for real-time updates, location search, and secure booking workflows.

Type: Labor Booking App

Role: Fullstack Developer

Tech-Stack: Flutter, Dart, Firebase, Node.js, MongoDB, Google Places API

Key Responsibilities & Achievements:

  1. ➤ Developed a mobile application to connect users with nearby labor service providers, enabling on-demand hiring through an intuitive interface.

    ➤ Built responsive and cross-platform UI using Flutter, ensuring seamless performance across Android and iOS devices.

    ➤ Implemented backend services with Node.js and MongoDB for managing user profiles, bookings, job categories, and availability.

    ➤ Integrated Google Places API to enable location-based service discovery and address auto-completion.

    ➤ Used Firebase for authentication, real-time updates, and push notifications to enhance user engagement and interaction.

    ➤ Designed and maintained secure APIs, ensuring efficient communication between frontend and backend systems.
